Intelligence Agriculture  Analysis and Recommendation System
========

Nowadays, there are a lot of different kinds of e-commercial business, and we are interested in analyzing these business behaviors. We want to find some methods to help the e-commercial companies to understand what merchandise customers like or dislike, and make decisions to improve the revenue. In this special subject, we predict whether customers reorder products or not in order to do precision marketing, and use association rules to obtain recommendation system of products.
